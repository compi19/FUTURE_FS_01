# Future Full-Stack Web Development Internship 🚀

This repository showcases the projects and technical tasks completed during my **Future Full-Stack Web Development Internship**. The curriculum focused on building modern, scalable web applications using the **MERN Stack** (MongoDB, Express.js, React, and Node.js).

## 📌 Internship Project Portfolio

Each task represents a milestone in my development as a Full-Stack Engineer, covering frontend design, backend logic, and database management.

### 1. [Personal Portfolio](./Task1_portfolio)
* **Overview:** A professional digital resume and project showcase designed to establish a personal brand.
* **Tech Stack:** React, Tailwind CSS, Lucide-react.
* **Features:** Responsive layouts, interactive UI components, and a clean professional aesthetic.
* **Key Learning:** Component-based architecture and advanced CSS styling with Tailwind.

### 2. [Client Lead Management (CLM) System](./Task2_CLM)
* **Overview:** A full-stack application for registering and tracking potential client leads.
* **Tech Stack:** MongoDB, Express.js, React, Node.js (MERN).
* **Key Features:** * Full CRUD (Create, Read, Update, Delete) functionality.
    * Bilingual support (Switch between Amharic and English).
    * Real-time lead status tracking and management.
* **Key Learning:** RESTful API development, database schema design, and handling complex state across the stack.

### 3. [Professional Corporate Website](./Task3_website)
* **Overview:** A high-performance, modern business landing page with service catalogs.
* **Tech Stack:** Vite, React, Tailwind CSS.
* **Features:** Optimized asset loading, smooth navigation, and service detail panels.
* **Key Learning:** Improving development speed with Vite and optimizing production builds.

## 🚀 Installation & Local Setup

To run any project from this repository on your local machine, follow these instructions:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/compi19/FUTURE_FS_01.git](https://github.com/compi19/FUTURE_FS_01.git)
    ```
2.  **Enter the specific task directory:**
    ```bash
    cd Task2_CLM
    ```
3.  **Install the required dependencies:**
    ```bash
    npm install
    ```
4.  **Launch the development server:**
    ```bash
    npm run dev # or 'npm start' depending on the package.json
    ```
